# Telegram RAG Bot

A development-only Telegram bot that ingests chat history and enables
semantic Q&A using a Retrieval-Augmented Generation (RAG) pipeline.

## Stack
- Telegram Bot API (Node.js)
- Python (Flask)
- Sentence-Transformers
- Supabase
- Hugging Face Inference API

## Setup
1. Create Telegram bot via BotFather
2. Add `.env` files
3. Run Python service
4. Run Node bot
